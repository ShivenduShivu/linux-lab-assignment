# Linux Graded Lab Assignment (Modules 1–4)

Hands-on Linux lab performed on **Ubuntu 22.04.5 LTS** (host `Senorita`).
Each folder contains the commands executed, their real outputs, the
required report file, and screenshots of the command executions.

## Repository Structure

| Folder | Question | Report file |
|--------|----------|-------------|
| `Question1_Environment_Verification` | Linux Environment Verification | `Environment_Report.txt` |
| `Question2_Secure_Workspace` | Secure Project Workspace Setup | `Project_Workspace_Report.txt` |
| `Question3_Link_Analysis` | File System and Link Analysis | `Link_Analysis_Report.txt` |
| `Question4_IO_Investigation` | File Access and I/O Investigation | `IO_Investigation_Report.txt` |
| `Question5_Storage_Assessment` | Storage Health Assessment (via vi) | `Storage_Assessment_Report.txt` |

Each folder also contains:
- `commands.txt` — the exact commands to run, in order
- `screenshots/` — screenshots of each command execution/output
- a `*_Report.txt` with **a 1–2 sentence explanation after every command**

## How this was produced
All commands were executed on a real Linux system and their genuine
outputs are recorded in the report files. Screenshots of each execution
are included in the per-question `screenshots/` folders.
